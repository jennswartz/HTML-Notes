Web and HTML Basic Concepts

Lesson 1 - The Basics of the Web and HTML

HTML - the basis for all web pages

HTTP - the protocol used between the internet and servers to communicate


HTML uses markup tags. Tags include:
  • Name
  • b (bold)
  • em (emphasis) = italics

HTML elements can have attributes. Examples include linking to websites like this.

Images use the img tag. Use alt="text " for blind screenreaders. Img is an example of a "void " tag, i.e. it does not have a closing tag.

Inline tag examples include a, span, br, img, and strong.

Block tag examples include div, p, and form. Block elements form an invisible box around them.

Lesson 1.2 - Creating a Structured Document

Overview of the Concepts:

  • HTML is the structure of the page;
  • CSS is the style;
  • JavaScript provides any interactivity.

"DOM " is Document Object Mode - meaning tree-like structure of a page.

"CSS " = Cascading Style Sheets - Used to provide color, font, etc.

Language --> syntax + rules

When designing webpages, cut the pages into boxes. For box, use div tags to define them.

Text editors - downloaded Sublime, which automatically adds tags. Use tags to create the trees that create the pages. CSS defines the styles.

Lesson 1.3 - Adding CSS Style and HTML Structure

CSS adds style to the page. It reads directions from the HTML code to apply style rules to format the page.

Design of webpages are made up of boxes. Think about all the boxes on the page while designing the page.

Developer tools in the browser can provide assistance for designing.

Avoid repetition. Give the same style to similar elements so you can modify them all at once by changing a single line of CSS.

Lesson 2.1 - Serious Programming

Important Concepts: Telling Computers What to Do

  • A computer can be programmed to do anything we want.
  • A computer program is a precise sequence of steps that a computer can follow to do something useful, e.g., browsers, games, mobile apps.
  • Programmers use a programing language, e.g., Python, to tell computers what to do.
  • Grammar - Must be precise or the program will not run.

Debugging – one of five ways that programmers think. Python code requires specific syntax, or code will not work. Must learn systems for debugging code.

Lesson 2.2 – Variables and Strings

Variables: a value that can change, depending on conditions or on information passed to the program. Variables are used to store information to be referenced and manipulated in a computer program. Variables also provide a way of labeling data with a descriptive name so that programs can be understood more clearly by the reader and ourselves. Think of variables as containers that hold information. Their sole purpose is to label and store data in memory. This data can then be used throughout a program. Expressions are any sequence of letters, numbers and underscores. Assign a value, then you can refer to it by name. Example: speed_of_light = 299792458. Variables can change if the values change.

A string is a sequence of characters. Must be in quotes or it will appear as a variable. You can index strings, select subsequences, and find strings within strings.

Lesson 2.3 – Input->Function->Output

Computers need an input, then perform a function (a/k/a procedure), then produce something else as an output.

Defining Functions: In Python, functions are defined by writing the 'def' keyword followed by a name, and then the 'parameters' for the function (inside parentheses), and a colon.

return vs print: You usually want to make sure to end your function with a return statement, otherwise they don’t really do anything.

Use inputs, functions, and outputs to build processes. Once you define a function once, you don't have to define it again.

Lesson 2.4 – Control Flow and Loops: If and While

There are two important programming tools to remember: decision making and repetition.

“if” statements control which code gets executed when; “while” loops make code perform the same task many times.

Loops use conditional statements. If the condition in the first statement of the expression is True, then the indented part of the expression is executed. Use if/else statements if you want want thing to happen if the expression is true, and another to happen if the expression is false.

The for-loop processes each item in a sequence. Each item in the sequence is assigned to the loop variable, and the body of the loop is executed. In other words, it loops through each item in a sequence. Example - use to print range 0 to 5 by looping through a sequence of numbers 0 to 5.

The while-loop keeps executing as long as the expression is true. It can go on forever. (Use sparingly because of this.) The while loop works by evaluating the condition in the initial expression to determine if it is true or false. If true, it executes the next statement, then it goes back to evaluate the condition again. If true, it keeps executing. It doesn't stop until the contion is false.

Lesson 2.5 – Debugging

There are five key debugging strategies

  1. Examine error messages when programs crash.
  2. Work from example code.
  3. Make sure examples work.
  4. Check, i.e., print, intermediate results.
  5. Keep and compare old versions.

Lesson 2.6 – Structured Data: Lists and For Loops

Lists and loops are ways to structure data. While we used a string is a sequence of characters, we can use list to sequence anything.

  • Lists are indexed just like letters in a word in a string.
  • Lists can be put inside of lists, i.e. nested within each other.

Mutation - Means changing the value of a list after its been created. Caution: if we change an assignment of one value, it could have impact if that value has an assignment.

Aliasing - When we have two different ways to refer to the same object.

Examples of List Operations

  • Append, which mutates the list it is invoked upon; it does not create a new list
  • Plus, which produces a new list (not a mutation of prior list); similar to concatenation of strings
  • Length, which gives the number of elements in a list also works on string (gives the number of characters)

Loops on Lists - Allow you to loop through elements in a list

Lesson 2.7 – How to Solve Problems

To solve a problem, you must understand it first. Use these strategies to solve problems:

  1. Don't panic.
  2. What are the inputs?
  3. What are the outputs?
  4. Work through some samples by hand.
  5. Use simple mechanical solutions.
  6. Develop solution incrementally and test as you go.
